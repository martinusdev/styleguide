§ 02 · Koncept

MCP = protokol
medzi AI a nástrojmi.

Open-source štandard od Anthropic (november 2024). Definuje, ako môže AI volať externé funkcie a získavať dáta — bez ohľadu na to, od koho model je.

Vývojár
Pýta sa
→
AI asistent
Claude, Cursor…
→
MCP Server
JSON-RPC
→
Styleguide
HTML / dáta
Ako USB-C. Zariadenia sa po zapojení najprv „pokecajú“ — čo vieš ty, čo viem ja: nabíjanie, dáta, obraz — a podľa toho nastavia spoluprácu. MCP robí to isté: server pri pripojení povie, aké nástroje ponúka a ako sa s nimi pracuje.

§ 04 · Stavebné bloky

MCP server ponúka tri veci.

Tools, resources, prompts. My zatiaľ používame len prvé.

Mapa = prompts · kniha = resources · náradie = tools

U nás 01 · Aktívne

Tools

Pasívne 02 · Dáta

§ 06 · Best practices

§ 07 · Kedy použiť

Kedy MCP, kedy nie.

Nie každý kontextový problém potrebuje vlastný server.

Vhodné

  • +
    Interný design system — čo robíme my
  • +
    Interné dáta a API
  • +
    Externé služby s kľúčom — Sentry, Slack
  • +
    Orientácia v kóde

Nevhodné

  • −
    Verejné API, ktoré AI pozná — Stripe, npm
  • −
    Čisté výpočty — regex, formátovanie
  • −
    Obrovské dáta
  • −
    Zápisy do produkcie bez človeka

  Heuristika: ak sa odpoveď dá vygúglovať, MCP netreba. Ak je za firewallom alebo sa mení každú hodinu — MCP áno. A MCP s API nesúperí — tool ich zvyčajne len obaľuje pre AI.

§ 10 · Architektúra

Jeden server, dva módy.

Rovnaký codebase — len iný zdroj dát.

External mode default · beží kdekoľvek

Server fetchuje aktuálne dáta z mrtns.sk/assets, drží ich v cache 5 minút. Vždy konzistentné s produkciou.

AI klient → MCP server → mrtns.sk
Internal mode dev · číta z repozitára

Aktivuje sa env premennou. Server číta YAML súbory priamo — zmeny vidno okamžite bez deploya. Navyše pridáva 3 interné tooly na čítanie SCSS a dát z repa.

AI klient → MCP server → app/*.yaml
§ 11 · Spustenie

Ako začať.

1

2

Nastav klienta

~10 riadkov do configu AI klienta. Hotový snippet nájdeš v docs/guides/mcp-server.md.

3

Hotovo

AI vidí komponenty, ikony aj utility — generuje rovno z nášho styleguide.
